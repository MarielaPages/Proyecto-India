INDIA PRODUCCIONES

Nosotros

INDIA es un equipo en el que se desarrolla, produce e implementa experiencias de marca. Combinamos estrategia y creatividad con tecnología de vanguardia para generar impacto en todo lo que hacemos. Nuestro objetivo es simple, producir ideas de forma estratégica, ágil y de máxima calidad.

PRODUCIMOS DE FORMA INTEGRAL

Servicios

Flipbook

Los invitados pasan por el set, se filman durante 7 segundos y en menos de 3 minutos tienen su librito de recuerdo.

Set Flip Book // Camara profesional HD Iluminación fotográfica // Flipbooks ilimitados Tapa y back personalizado.

Foto Croma

Los invitados pasan por el set, eligen un fondo para su foto y se la llevan impresa con marco personalizado.

Set Croma con tótem ploteado // Camara profesional HD e iluminación // Impresiones ilimitadas 10x15 cm Hasta 4 fondos de foto // Marco e interfaces personalizadas

Face Match

Los invitados deberán imitar una foto que aparece en pantalla. Al finalizar, se llevarán una impresión con ambas versiones

Photobooth ploteado // Camara profesional HD Impresiones ilimitadas 10x15 cm // Diseño de marco de foto personalizado // Entrega de material en digital

Gif Animado

Los invitados pasan por tótem para realizar un gif que recibirán por mail o whatsapp.

Tótem Photobooth ploteado // Camara profesional HD GIF animado // Marco e interfaces personalizadas Envío por Whatsapp o email.

Photobooth

Los invitados posan frente al tótem que les sacará una, dos o tres fotos. La misma se imprime con marco personalizado.

Photobooth ploteado // Camara profesional HD Impresiones ilimitadas 5x15 cm Diseño de marco de foto personalizado // Interfaces personalizadas.

Mirror Booth

Los invitados posan frente al espejo que les sacará una foto. La misma se imprime en 10x15cm con marco personalizado.

Mirror Booth // Camara profesional HD + Flash Impresiones ilimitadas 10x15 cm Diseño de marco de foto personalizado // Interfaces personalizadas.

Foto Stickers

Los invitados podrán personalizar su foto con stickers digitales a través de una pantalla táctil.

Tótem Photobooth ploteado // Camara profesional HD Stickers digitales // Impresiones ilimitadas 10x15 cm Marco e interfaces personalizadas.

Neon Booth

Cabina Neon // Camaras HD Sesiones ilimitadas // Foto impresa o GIF animado. Entrega de material en digital.
